# 💐 Sorpresa Romántica Web para tu Novia  

**¡Una página interactiva con flores animadas y música personalizable!**  

---

## 🛠️ Configuración rápida  
1. Clona/descarga este repositorio.  
2. Abre `index.html` en tu editor de código.  

---

## 🎵 Cambiar la música  
Reemplaza el archivo de audio en `/assets/audio/`:  

```html
<!-- En el <head> de index.html -->  
<audio id="musica" loop>  
    <source src="assets/audio/musica.mp3" type="audio/mp3">  
</audio>
